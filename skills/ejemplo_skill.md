# Plantilla de Skill

## Nombre de la Skill
<!-- Un nombre corto y descriptivo para la skill. -->
Ejemplo: `analizador_de_codigo`

### Descripción
<!-- Una explicación clara de lo que hace esta skill y cuándo debe usarse. -->
Esta skill analiza un fichero de código y proporciona un resumen de su complejidad y posibles errores.

### Comandos / Flujo de trabajo
<!-- Los pasos o comandos que la skill ejecuta. -->
1. Leer el contenido del fichero.
2. Contar el número de líneas, funciones y clases.
3. Identificar patrones de código complejos (ej. bucles anidados).
4. Generar un informe en formato Markdown.

### Ejemplos de uso
<!-- Un ejemplo de cómo un usuario o el agente invocaría esta skill. -->
"Opencode, usa la skill 'analizador_de_codigo' en el fichero 'main.py'."
